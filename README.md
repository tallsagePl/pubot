# Push-up Telegram Bot

Telegram-бот для учета отжиманий с персональным прогрессом каждого пользователя.

## Возможности

- отдельный прогресс для каждого пользователя
- `/start <цель>` - задать дневную цель
- `/add <число>` - добавить выполненные отжимания
- `/left` - сколько осталось на сегодня
- `/record` - лучший дневной результат за все время
- `/allfrom` - сумма отжиманий с даты или с начала
- перенос недовыполненных отжиманий на следующий день
- день обновляется в `04:00` (по таймзоне `BOT_TIMEZONE`)
- напоминания в `13:00`, `16:00`, `19:00`, `22:00` если цель не выполнена
- данные сохраняются в `data/users.json`, не теряются после перезапуска

## Запуск

1. Установи зависимости:
   - `npm install`
2. Создай `.env` из `.env.example` и задай токен:
   - `TELEGRAM_BOT_TOKEN=...`
3. Запусти:
   - `npm start`

## Команды

- `/start 100`
- `/add 25`
- `/left`
- `/record`
- `/allfrom`
